## 🛠️ 核心功能模块（12大AI工具）

### 1️⃣ 📋 AI需求生成器
**一键生成符合大四学生水平的毕业设计功能需求文档**

- 🎯 输入项目主题和核心功能，AI自动生成完整需求文档
- 📝 包含功能描述、用户角色、系统功能清单、非功能性需求等
- 🎓 主打真实、不虚浮，符合实际学生水平
- 💾 支持复制和下载为TXT文档

**适用场景**：开题报告、需求分析文档撰写、答辩准备

### 2️⃣ 🗂️ 表结构文档生成
**智能生成规范的数据库表结构设计文档**

- 📊 输入项目描述，AI自动设计数据库表结构
- 📐 包含表名、字段名、数据类型、约束、索引、备注等完整信息
- 📄 支持导出Word文档，格式规范专业
- 🔍 自动生成主键、外键、索引设计建议

**适用场景**：数据库设计文档、系统设计说明书、答辩展示

### 3️⃣ 🔗 多表ER图生成
**自动生成标准的实体关系图（多表关联）**

- 🎨 基于Mermaid渲染，支持复杂的多表关系展示
- 🔄 自动识别一对一、一对多、多对多等关系
- 🖼️ 支持导出PNG、SVG格式
- 📋 提供Mermaid源码下载，便于二次编辑

**适用场景**：数据库设计、系统设计文档、答辩PPT

### 4️⃣ 📊 单表ER图生成
**快速生成单个数据表的ER图**

- 🎯 专注单表字段展示，结构清晰
- 📐 自动标注主键、外键、唯一索引
- 🖼️ 支持PNG/SVG导出和代码下载
- 💡 适合小型项目或模块化展示

**适用场景**：单模块数据库设计、局部设计展示

### 5️⃣ 🧪 测试数据生成器
**智能生成符合业务逻辑的测试SQL数据**

- 🤖 根据表结构和业务逻辑，自动生成合理的测试数据
- 📝 生成可直接执行的SQL INSERT语句
- 🔢 支持批量生成（10-100条数据）
- ✅ 自动处理外键关联、数据约束、枚举值等

**适用场景**：系统测试、功能演示、答辩数据准备

### 6️⃣ 📈 AI流程图生成器
**快速生成标准化的业务流程图**

- 🎨 基于Mermaid绘制，支持复杂流程逻辑
- 🔄 包含条件判断、循环、并行等流程元素
- 🖼️ 支持PNG/SVG导出
- 🔍 支持鼠标滚轮缩放查看细节

**适用场景**：业务流程分析、系统功能说明、答辩演示

### 7️⃣ ⏱️ AI时序图生成器
**自动生成系统交互时序图**

- 📡 展示对象之间的消息传递和时间顺序
- 🎯 适合描述用户-前端-后端-数据库交互流程
- 🖼️ 支持导出PNG/SVG和Mermaid源码
- 🔍 支持缩放和详细查看

**适用场景**：接口设计文档、系统交互说明、技术答辩

### 8️⃣ 🏗️ AI类图生成器
**生成面向对象的UML类图**

- 🎓 展示类的属性、方法、继承、关联关系
- 📐 符合UML规范，适合Java/C++/Python等OOP项目
- 🖼️ 支持导出多种格式
- 💡 自动识别设计模式和类关系

**适用场景**：系统设计文档、代码架构说明、面向对象设计

### 9️⃣ 👥 AI用例图生成器
**生成UML用例图，展示系统功能和角色**

- 🎭 清晰展示系统参与者（用户、管理员等）和用例
- 🔗 自动识别用例之间的包含、扩展关系
- 🖼️ 基于PlantUML渲染，支持SVG导出
- 🔍 支持鼠标滚轮缩放功能

**适用场景**：需求分析、系统功能展示、答辩演示

### 🔟 🏛️ AI架构图生成器
**生成系统架构设计图**

- 🏗️ 展示系统的分层架构（前端、后端、数据库、中间件等）
- 📊 包含技术栈选择、模块划分、部署架构等
- 🖼️ 支持SVG导出和PlantUML源码下载
- 🔍 支持鼠标滚轮缩放查看

**适用场景**：系统设计文档、技术选型说明、答辩展示

### 1️⃣1️⃣ 🧩 AI功能结构图生成器
**生成系统功能模块结构图**

- 📦 清晰展示系统的功能模块划分和层次关系
- 🎯 适合展示系统的功能组织结构
- 🖼️ 基于D3.js渲染，支持交互式展开折叠
- 💾 支持PNG导出

**适用场景**：系统功能说明、模块划分展示、需求分析

### 1️⃣2️⃣ 🧠 AI思维导图生成器
**生成项目思维导图，梳理项目思路**

- 🌳 树状结构展示，层次清晰
- 🎨 基于Markmap渲染，支持Markdown语法
- 🔄 支持节点展开折叠，便于聚焦关键内容
- 🖼️ 支持PNG导出和Markdown源码下载

**适用场景**：项目规划、知识梳理、答辩准备
